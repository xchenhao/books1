    # 模块

    是指在代码变得相当大之后，为了将需要重复使用的有组织的代码段放在一起，这部分代码可以附加到现有的程序中，附加的过程叫做导入（import）

    导入模块的一般写法

    import 模块名称
from 模块名称 import 方法名

    # 异常

    错误 != 异常

    • 是指在出现错误时采用正常控制流以外的动作 = 一般处理流程：检测到错误，引发异常；对异常进行捕获的操作
    try:
    <监控异常>
except Exception[, reason]:
    <异常处理代码>
finally:
    <无论异常是否发生都执行>
